ZZ 保存并退出文件
hjkl 左下上右移动（前边加数字，移动多次3h想左边移动3位）
w 光标向前移动一个Word，b 向后移动一个word（支持计数器） ps.计数器即前边加数字
$ 跳到行尾 ^跳到行首
命令“fx”,f 意为find，光标从当前位置移动到从当前开始第一个x的位置
命令“tx”,t 意为to,与“fx”功能相反
命令“%”，跳到“（”匹配的“）” ps。[] {} 同样适用
命令“gg”，跳至第一行；G 跳至末尾行。当你用"G"命令从一个地方跳转到另一个地方时， Vim 会记得你起跳的位置 。使用“``”跳回到刚才的出发点：
命令“%”前指定一个命令计数，比如"50%"将会把光标定位在文件的中间，"90%"跳到接近文件尾的地方
当前显示的界面中，H M L 分别跳至开头、中间、最后。H home M middle  L last
使用crtl-g，显示一些信息
set ruler，在 Vim 窗口的右下角显示当前光标位置
set hlsearch 设置搜索结果高亮， set nohlsearch 反之

"dw"命令可以删除一个 word. 你可以把其中的"w"看作是向右移一个 word 的命令 。 事实上，
 "d"命令可以后跟任何一个位移命令，它将删除从当前光标起到位移的终点处的文本内容 。
用替换命令 s/expr// 只对光标所在行有效。  %s/expr//对全文有效  （expr正则匹配）
用J删除换行符,及合并行
用u撤销命令，U撤销一行的操作，crtl+R取消撤销
用":e!"命令放弃所有修改并重新载入该文件的原始内容
o当前行的下面另起一行,O 当前行的上面另起一行
vim -x test 为新编辑的文件加密,停止对该文件加密‘set: key=’ 或者‘:X’,普通用户的文件可以被root权限查看
